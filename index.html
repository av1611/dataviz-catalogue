Times Visual Vocabulary

Instructions

Select one of the charts above to go to its demo page.

All the code necessary to render the charts stand-alone is in the corresponding folder.

Install the SVG Crowbar to download the chart as an SVG. This SVG file can be opened in Adobe Illustrator or sent to the Graphics desk.

You install the Crowbar by dragging the bookmarklet on the page to your bookmark bar. Then open one of the charts and click on the bookmarklet to download.

You can modify the code and tweak the chart by playing with the files themselves. You can also copy-paste the code to a BlockBuilder to generate a chart you can then download.
