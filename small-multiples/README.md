## Data format

```
[
  {
    date: 2008,
    value: 10,
    facet: "foo"
  },
  {
    date: 2009,
    value: 20,
    facet: "foo"
  },
  {
    date: 2008,
    value: 1,
    facet: "bar"
  },
  {
    date: 2009,
    value: 10,
    facet: "bar"
  },
]
```
